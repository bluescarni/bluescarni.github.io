Detailed Description

template<typename Cf, typename Key>
class piranha::term< Cf, Key >

Term class.

This class represents series terms, which are parametrised over a coefficient type Cf and a key type Key. One mutable coefficient instance and one key instance are the only data members and they can be accessed directly.

Type requirements

  • Cf must satisfy piranha::is_cf.
  • Key must satisfy piranha::is_key.

Exception safety guarantee

This class provides the strong exception safety guarantee for all operations.

Move semantics

Move semantics is equivalent to its data members' move semantics.

Serialization

This class supports serialization if the coefficient and key types support it.
