Detailed Description

template<typename Key, typename T>
class piranha::key_is_evaluable< Key, T >

Type trait to detect evaluable keys.

This type trait will be true if Key is a key type providing a const method evaluate() taking a const instance of piranha::symbol_set::positions_map of T and a const instance of piranha::symbol_set as input, false otherwise. If Key does not satisfy piranha::is_key, a compilation error will be produced.

The decay type of Key is considered in this type trait.
