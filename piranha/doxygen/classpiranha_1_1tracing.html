Detailed Description

Tracing class.

This class is used to trace events for analysis and statistical purposes. It is comprised of static methods that allow to interact at runtime with a database of events in which string descriptors are paired with arbitrary data.

All methods in this class are thread-safe and provide the strong exception safety guarantee, unless otherwise specified. Methods and data members of this class are static, hence this class provides trivial move semantics.

inlinestatic

Trace event.

Trace an event uniquely identified by str, and apply the functor f to the data associated to it.

Functor is expected to be an unary function object taking as parameter the reference to a boost::any instance representing the data associated to the descriptor str. The first time a descriptor is used, the boost::any object passed to f will be empty.

If tracing is disabled in piranha::settings, this method will be a no-op.

Parameters
[in]strthe descriptor of the event being traced.
[in]fthe functor to be applied to the data associated to str.
Exceptions
unspecifiedany exception thrown by:
  • copy construction of std::string,
  • memory errors in standard containers,
  • failures in threading primitives,
  • the execution of functor f.
See also
http://www.boost.org/doc/libs/release/libs/any
